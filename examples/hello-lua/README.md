# hello-lua

This example uses [Emscripten](https://emscripten.org/) to compile C code to WebAssembly.

## Prerequisites

Get the Emscripten SDK, using these instructions:

https://emscripten.org/docs/getting_started/downloads.html

## Run the example

```console
yarn install
yarn serve
```
